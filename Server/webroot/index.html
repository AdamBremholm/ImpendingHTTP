Impending HTTP-Server.
Av Adam Bremholm, Patrik Bågenholm och Filip Christofersson.

Servern kan hantera JSON i POST och GET, för att returnera som JSON eller spara till databas.
URL kan läsas som /json?key=value&key2=value2, osv.
Databasen är en remote MongoDB, men det är möjligt att ändra till en List Storage.
Kan hantera filer från med rätt content-type
(PDF och PNG från länkarna, HTML/CSS visas, och JS är Javascript-knappen)
Plugins som finns: DB upsert/find, statistik om requests till servern, multiplicera två tal, DynamicWePage, SavePerson med html som payload.
